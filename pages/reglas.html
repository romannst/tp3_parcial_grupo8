El juego de memorizar

Reglas del juego

El objetivo del juego es encontrar la mayor cantidad de pares de tarjetas iguales una vez que se hayan dado vuelta todas las tarjetas. Para jugar, sigue estas reglas:

  1. Al iniciar el juego, las tarjetas se mostrarán boca abajo.
  2. Hacer clic en una tarjeta para dar vuelta y revelar su imagen.
  3. Luego, se hace lo mismo con las demás tarjetas para intentar encontrar la pareja correspondiente.
  4. Si las tarjetas coinciden, se mantienen boca arriba. Si no coinciden, se ocultan nuevamente después de un breve período de tiempo.
  5. El juego sigue hasta que se hayan encontrado todas las parejas de tarjetas.
  6. Tu puntaje refleja el número de aciertos y el tiempo que te haya tomado completar el juego.
